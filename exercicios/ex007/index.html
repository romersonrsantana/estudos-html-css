Principais formatações

Negrito/destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântico), simplismente você escolheu destacar o termo.

Nesta frase, tenho um termo em termo em destaque usando a tag <strong> (semântico), neste caso o termo em destaque é forte, carregado de sentido.

Itálico / Ênfase

Nesta frase temos um termo em Itálico usando a tag <i> (não semântica) simplismente uma forma em itálico.

Nesta frase temos um termo em ênfase usando a tag <em> (semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>;

E neste parágrafo temos porém com o marcador de outra cor; usando a tag <style>;

Texto com letras pequenas

Esta teg é usada semanticamente para dar sentido as letras pequenas, ex.: contratos; tag <small>;

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag <del>;

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi inserido depois. Usando a tag <ins>;

Texto sobrescrito

Para inserir coisas do tipo x20+3. Usamos a tag <sup>;

Texto sub-escrito

Para inserir coisas do tipo H2O. Usamos a tag <sub>;
