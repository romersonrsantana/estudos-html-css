Seletores personalizados

Para evitar ou melhor diferenciar uma formatação que englobe todas as marcas "h1" ou todos os parágrafos, podemos colocar um id ou class nas tags.

id

Segundo a W3C por regra não podemos utilizar mais de um id em um mesmo documento.

class

Já a class pode ser usada por várias vezes em um mesmo documento como forma de identiicador.
