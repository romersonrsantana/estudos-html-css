Configurações Globais nas CSS

Nas conigurações globais é usado o asterisco com as chaves e dentro é adicionado as configurações que vão servir para todo o HTML *{configurações}.

Configurações no plano de fundo da página

Para um degradê no plano de fundo da página é utilizado a marcação com o seletor:
body{background-image: linear-gradient(to right,cor01, cor02)}

Configurações dentro do main{}

Dentro das CSS é possível colocar o main dentro da tag style, nesse caso o main passa a funcionar como uma div, se tornando a parte principal da página,
segue algumas configurações que podemos colocar dentro do main:

  1. Background-color:
    Cor de fundo da div.
  2. Border-radius:
    Formato dos cantos da div.
  3. Width
    Largura
  4. Padding
    Acochoamento do conteúdo dentro da div.
  5. Margin
    Centraliza o título

Box-Shadow

Podemos dizer que essa tag transforma o plano de fundo da div, usando como base três parãmetros:

  1. Quanto a sombra vai andar para o lado;
  2. Quanto de espalhamento ela vai ter;
  3. Qual será a cor e o quanto de transparência;
